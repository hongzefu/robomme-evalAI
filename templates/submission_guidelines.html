Upload a single JSON file with exactly one field:

{
  "agent_url": "https://your-agent.example.com"
}

agent_url must be an http or https base URL. EvalAI will call {agent_url}/act during evaluation.

The manifest is rejected if:

  • the JSON is invalid or contains extra fields
  • the URL is missing a hostname or includes a path, query, or fragment
  • the URL resolves to loopback or private network addresses

Your agent service must stay reachable for the full evaluation window and must answer every POST /act request within the configured timeout.
