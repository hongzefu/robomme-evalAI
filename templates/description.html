This challenge evaluates a remotely hosted HTTP agent on MiniGrid-Empty-5x5-v0. EvalAI does not run participant code inside a container. Instead, each submission uploads a JSON manifest that points to a live agent service, and the evaluator repeatedly sends observations to POST /act.

The evaluator creates the environment locally and validates the HTTP agent protocol on fixed-seed episodes for each split. Leaderboard sync now uses an 18-column placeholder task contract: BinFill, PickXtimes, SwingXtimes, StopCube, VideoUnmask, VideoUnmaskSwap, ButtonUnmask, ButtonUnmaskSwap, PickHighlight, VideoRepick, VideoPlaceButton, VideoPlaceOrder, MoveCube, InsertPeg, PatternLock, RouteStick, SuccessRate, and OverallSuccessRate.

This repository revision does not implement real task scoring yet. All 18 leaderboard columns are currently emitted as placeholder 0.0 values so the EvalAI challenge configuration, remote worker, and submission payloads stay in sync. The public leaderboard is still based on the test phase public split; the private split remains hidden for final ranking and auditing.
